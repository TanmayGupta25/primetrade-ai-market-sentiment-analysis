# 📈 PrimeTrade.ai Market Sentiment Analysis
### Exploring the Relationship Between Bitcoin Market Sentiment and Hyperliquid Trader Performance

> **A comprehensive end-to-end Data Science project submitted as part of the PrimeTrade.ai Data Science Hiring Assignment.**

---

## 📌 Project Overview

Financial markets are heavily influenced by investor sentiment. This project investigates whether **Bitcoin Market Sentiment (Fear & Greed Index)** has a measurable relationship with **trader behavior and performance** on the Hyperliquid trading platform.

Using statistical analysis, exploratory data analysis (EDA), and business intelligence techniques, this project uncovers patterns in trading activity, profitability, leverage usage, and risk under different market sentiment conditions.

---

## 🎯 Objectives

- Analyze Bitcoin Market Sentiment (Fear & Greed Index).
- Explore Hyperliquid trader behavior and performance.
- Integrate both datasets using date-based matching.
- Investigate relationships between market sentiment and:
  - Trader profitability
  - Win rate
  - Leverage usage
  - Trade size
  - Trading activity
- Validate findings using statistical hypothesis testing.
- Present insights through executive dashboards and business recommendations.

## 📂 Dataset Information

### 1. Bitcoin Market Sentiment Dataset
Contains daily Bitcoin market sentiment classified using the Fear & Greed Index.

**Key Features**

- Sentiment Value

### 2. Hyperliquid Historical Trading Dataset

- Trader Account

- Execution Price
- Trade Size
- Leverage

- Timestamp
